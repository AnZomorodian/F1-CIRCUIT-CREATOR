# F1 Track Designer 🏎️

![F1 Designer Hero](client/src/assets/hero.jpg)

An advanced, interactive web application for designing professional Formula 1 circuits.

## 🚀 Quick Start
```bash
npm install
npm run dev
```

## 🛠 Features
- **Canva-style Editor**: Drag and drop track pieces onto a high-performance canvas.
- **Magnetic Snapping**: Objects connect perfectly without gaps.
- **Customization**: Full control over geometry, curbing, and positioning.
- **Community Library**: Save your tracks and browse creations from others.
- **Modern UI**: Sleek dark mode interface designed for performance.

## 📦 Project Structure
- `/client`: React + Vite frontend
- `/server`: Express + Node.js backend
- `/shared`: Shared types and API routes
- `DATABASE.JSON`: Local storage for user data
